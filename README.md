# Java Encryption GUI Project 
This project demonstrates a secure communication system built using Java Swing and custom hybrid encryption. 
It includes both encryption and decryption features with user-defined keys for enhanced security. 

## Features
- Hybrid Encryption & Decryption
- Java Swing-based GUI
- Copy/Share Encrypted Text

## How to Run
1. javac EncryptionApp.java
2. java EncryptionApp
